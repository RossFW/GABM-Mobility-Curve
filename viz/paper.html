4. Methods

  • 4.1 Simulation setup. Dewberry Hollow town; single decision per agent per condition (stay home / go out).
  • 4.2 Agent population (Fig 22). 100 frozen personas with Big Five traits, gender, age. Sampled to balance trait combinations.
  • 4.3 Probe design. 40 infection levels (0%–7%) × 5 repetitions per (agent, level) cell.
  • 4.4 LLM configurations. 21 configs spanning OpenAI, Anthropic, Google: provider, model size, generation, reasoning budget. Cost / token bookkeeping summarized.
  • 4.5 Response collection. Structured JSON outputs (decision + reasoning text). 0 format errors across 420,000 responses.
  • 4.6 Analytical methods.
    • Mobility curves: mean stay-home rate per infection level per config; OLS dummy-variable comparisons.
    • Logistic regression: fixed-effects (infection-curve only) and random-effects (with traits, gender, age, random intercept per agent). Diagnostics in Appendix C.
    • Logistic regression with mention flags: extends random-effects logit with yes/no flags for whether the response mentions each trait label or context category.
    • Spearman ρ: rank-based agreement on predictor importance and on per-agent stay-home rates.
    • Embedding cosine similarity: OpenAI text-embedding-3-large; persona individuation via within- vs. across-agent same-decision pairs.
    • Mention keyword construction and validation.
